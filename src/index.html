Here are a collection projects that I have worked on.

  • Home Control

    Home Control is a configurable automation service that works with Zigbee2MQTT in order to control smart home devices locally.

    Written in Typescript / Express / Pug, tested with Jest / Github Actions & interfaces with MQTT.

    https://github.com/jamie6king/home-control

  • Scam Radar

    Scam Radar is a Chrome extension / service that would scan a Ebay listing for a car, compare the information to official DVLA / DVSA data, and inform the user of any discrepancies.

    Written in Javascript / Express, with a Manifest V3 extension & service interfacing with MongoDB, tested with Jest / Github Actions.

    https://github.com/jamie6king/scamradar

  • SkyCast

    SkyCast was a React application that displayed in-depth weather information about a location, using a nice layout & fancy graphics.

    Written in React / Express / Javascript, using various third-party libraries & uses the OpenWeatherMap API.

    https://github.com/jamie6king/skycast

  • LuaSync

    LuaSync is a Lua-based file syncing utility, which uses Rclone & inotifywait to automatically sync files to a remote.

    https://github.com/jamie6king/luasync

  • Personal Website

    A personal website showing my projects & contact information. Statically written in HTML / CSS, using modern features, such as nesting, scroll-snap & variables.

    https://github.com/jamie6king/website
